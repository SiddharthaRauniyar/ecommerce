
# 🛒 EcoSecure Ecommerce Project

A secure full-stack Django-based e-commerce web application with admin dashboard, order management, invoice generation, and order tracking system.

---

## 🚀 Project Overview

EcoSecure is a modern e-commerce platform designed with a focus on:
- Secure development practices
- User-friendly shopping experience
- Admin-controlled order management
- Order tracking system with status updates
- Invoice generation system

---

## ✨ Features

### 👤 User Features
- User registration and login system
- Product browsing and search
- Shopping cart functionality
- Checkout system
- Order placement
- Order history
- Order tracking system
- Invoice download/print

---

### 🛠 Admin Features
- Admin dashboard with analytics
- Product management (add/edit/delete)
- Order management system
- Order status updates (pending → shipped → delivered)
- Revenue tracking

---

### 🔐 Security Features
- User authentication system
- Session security
- CSRF protection
- Input validation
- Secure email handling using environment variables

---

### 📦 Order System
- Order creation at checkout
- Order status workflow:
  - Pending
  - Processing
  - Packed
  - Shipped
  - Delivered
  - Cancelled
- Order history tracking

---

### 📊 Tracking System
- Real-time order status updates
- Admin-controlled tracking updates
- Order timeline history

---

## 🧰 Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite3
- Authentication: Django Auth System
- Email Service: SMTP (Gmail)
- Version Control: Git & GitHub
